# Software-Testing
Meu objetivo é registrar aprendizados em sala de aula sobre Teste de Software.

## 📚Aprendizados:
  - **Teste Unitário:** Verifica o funcionamento correto duma parte do código, seja método ou função.
    O teste pode ser feito de forma **manual**,-validando e testando cada item observando o output sem ferramenta/script automatizado-, e, de forma **automatizada**,-usando ferramenta/script para testar rapidamente e a cada vez que o código mudar-.
